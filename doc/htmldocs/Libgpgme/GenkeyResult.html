Fingerprint Property

A fingerprint of the key that was created.

Syntax

public string Fingerprint { get; }

Value

A string that contains the (hexdecimal) fingerprint of the key that was created.

Remarks

If both a primary and a sub key were generated, the fingerprint of the primary key will be returned. If the crypto engine does not provide the fingerprint, GenkeyResult.Fingerprint will be null.
